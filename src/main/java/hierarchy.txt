Project Options Hierarchy with Comprehensive Step-by-Step Processes

1. Main Menu (Primary application hub for navigating between modules)
   ├── 🏫 1. Study (University and academic management menu)
   │   ├── 1. Attendance Options (Manage and track course attendance records)
   │   │   ├── 1. Mark Today's Attendance (Process: [Select Course from List] -> [Select Instructor if multiple] -> [Enter Time: HH.MM followed by a/p or 'no'] -> [Mark Presence: P for Present, . for Absent] -> [Enter number of signatures taken] -> [Enter Topic or Skip])
   │   │   ├── 2. Edit Attendance Info Menu (Sub-menu to modify existing attendance data)
   │   │   │   ├── 1. Edit Topic (Process: [Select Date] -> [Select Course] -> [Select Teacher] -> [Enter New Topic])
   │   │   │   ├── 2. Edit Online/Offline (Process: [Select Date] -> [Select Course] -> [Select Teacher] -> [Toggle Online/Offline])
   │   │   │   ├── 3. Edit Attendance Status (Process: [Select Date] -> [Select Course] -> [Select Teacher] -> [Toggle Present/Absent])
   │   │   │   ├── 4. Edit Attendance Count (Process: [Select Date] -> [Select Course] -> [Select Teacher] -> [Enter New Signature Count])
   │   │   │   ├── 5. Study Materials (Book/Sheet) (Process: [Select Date] -> [Select Course] -> [Select Teacher] -> [Enter Book/Sheet Name or URL])
   │   │   │   └── 0. Back (Return to Attendance Options)
   │   │   ├── 3. Extra Class (Process: [Enter Date: DD-MM-YYYY] -> [Select Course] -> [Select Instructor] -> [Enter Time: HH.MM + a/p] -> [Enter Presence: P/.] -> [Enter Signature Count] -> [Enter Topic] -> [Enter Number of Extra Classes])
   │   │   ├── 4. Mark Missed Attendance (Process: [Enter Missed Date: DD-MM-YYYY] -> [Select Course] -> [Select Instructor] -> [Enter Time: HH.MM + a/p] -> [Mark Presence] -> [Signature Count] -> [Enter Topic])
   │   │   ├── 5. Special Attendance (Process: [Select 'All Courses' or Specific Course] -> [Enter Start Date] -> [Enter End Date] -> [Skip Friday/Saturday? Yes/No] -> [Enter Topic] -> [Select Instructor for each course] -> Automatically marks as Present)
   │   │   ├── 6. Attendance Details & Reports (View detailed logs and statistics)
   │   │   │   ├── 1. All Attendance Records (Displays chronological list of every attendance entry)
   │   │   │   ├── 2. Current Semester Attendance Report (Summary percentage and count for active semester)
   │   │   │   ├── 3. Calendar Attendance (Process: [Select Semester] -> [Select Course] -> [Select Month] -> Shows grid of dates marked)
   │   │   │   ├── 4. Online Class Details (Shows a list of all classes marked as 'Online')
   │   │   │   ├── 5. Generate Attendance Calendar PDF (Process: [Select Semester] -> [Generates PDF and opens it])
   │   │   │   └── 0. Back (Return to Attendance Options)
   │   │   ├── 7. Remove Attendance Record (Process: [Select Date] -> [Select Course] -> [Select Instructor] -> [Confirm Deletion])
   │   │   └── 0. Back to University Menu (Return to Study Menu)
   │   ├── 2. Done Study Management (Track completed course topics)
   │   │   ├── 1. Mark Class Sessions as Done Study (Process: [Select Semester] -> [Select Course] -> [Select Teacher] -> [Select Attendance Entry to mark as Studied])
   │   │   ├── 2. View Overall Course Study Progress (Shows % of studied topics vs total classes for all courses)
   │   │   ├── 3. View Individual Course Study Progress (Process: [Select Course] -> [Displays detailed topic list with Study Status])
   │   │   ├── 4. View All Course Details (Summary table of all classes, teachers, and study progress)
   │   │   ├── 5. Pending Study Details (Process: [Select Course] -> [Displays only topics marked as 'Not Studied'])
   │   │   ├── 6. Attendance History (with Topics) (Chronological list of classes showing the taught topic for each)
   │   │   ├── 7. Remove Done Study Mark (Process: [Select Semester] -> [Select Course] -> [Select Teacher] -> [Select entry to revert to Not Studied])
   │   │   └── 0. Back to Study Menu (Return to Study Menu)
   │   ├── 3. Configure Course Section (Process: [Select Semester] -> [Enter Start Date] -> [Enter End Date] -> [Loop: Enter Course Code, Title, and Teachers until 'done'])
   │   ├── 4. Make Current Semester (Process: [Select from list of semesters L1S1-L4S2] -> [Updates active context])
   │   ├── 5. Set Course Details (Manage marks and credits)
   │   │   ├── 1. Add Course (Process: [Select Semester] -> [Enter Code] -> [Enter Title] -> [Enter Teachers])
   │   │   ├── 2. Enter Particular Course to Edit Marks/Credits (Process: [Select Course] -> [Enter Credit Hours] -> [Enter Mid Marks] -> [Enter Final Marks] -> [Enter Assignment Marks])
   │   │   ├── 3. Delete Course (Process: [Select Semester] -> [Select Course Code] -> [Confirm Removal])
   │   │   └── 0. Back to University Menu (Return to Study Menu)
   │   ├── 6. CGPA Calculation (Performance tracking)
   │   │   ├── 1. View Semester SGPA (Process: [Select Semester] -> [Calculates based on entered Course Marks and Credits])
   │   │   ├── 2. View Overall CGPA (Combines all SGPA results from all semesters)
   │   │   ├── 3. Clear Semester CGPA Data (Process: [Select Semester] -> [Wipes SGPA for that semester])
   │   │   ├── 4. Clear All CGPA Data (Wipes all academic records)
   │   │   └── 0. Back to University Menu (Return to Study Menu)
   │   ├── 7. My Details (Process: [Enter Name] -> [Enter Faculty] -> [Enter Student ID] -> [Enter Reg. No] -> [Enter Session])
   │   ├── 8. Registration Fee Management (University finance tracking)
   │   │   ├── 1. Add New Registration Fee (Process: [Select Semester] -> [Enter Fee Title] -> [Enter Amount] -> [Enter Date: DD-MM-YYYY or 't'])
   │   │   ├── 2. Edit Registration Fee (Process: [Select Semester] -> [Select Fee Entry] -> [Edit Title/Amount/Date])
   │   │   ├── 3. Remove Registration Fee (Process: [Select Semester] -> [Select Fee Entry] -> [Confirm Deletion])
   │   │   ├── 4. View Registration Fees for Semester (Process: [Select Semester] -> [Shows list of fees and dates])
   │   │   ├── 5. View Total Fees (Semester) (Process: [Select Semester] -> [Sum of all fees for that term])
   │   │   ├── 6. View Overall Total Fees (Grand total sum across all semesters)
   │   │   ├── 7. Generate Semester Fee Report (PDF) (Process: [Select Semester] -> [Generates PDF])
   │   │   ├── 8. Generate Overall Fee Report (PDF) (Generates PDF of all fee history)
   │   │   └── 0. Back (Return to Study Menu)
   │   ├── 9. Generate University Reports (PDF) (Central academic export menu)
   │   ├── 10. Semester Details (Process: [Select Semester] -> [View sub-details for that term])
   │   │   ├── 1. View Class Details (Process: [Select Semester] -> [Lists all classes with topics and teachers])
   │   │   ├── 2. View Marks Details (Process: [Select Semester] -> [Table of Course Code vs Mid/Final/Assignment marks])
   │   │   └── 0. Back to University Menu (Return to Study Menu)
   │   ├── 11. Comprehensive University Data (Full summary view of academic data)
   │   └── 0. Back to Main Menu (Return to Main Menu)
   ├── 📅 2. Manage Daily Routine (Task management)
   │   ├── 1. Add New Task (Process: [Select Category: Exam/Assignment/Study/etc.] -> [Enter Task Name] -> [Enter Start Time: HH.MM + a/p] -> [Enter End Time: HH.MM + a/p] -> [Enter Date: DD-MM-YYYY or 't'])
   │   ├── 2. View Pending Tasks (Lists all tasks with status 'Pending' or 'In Progress')
   │   ├── 3. View Tasks for Today (Filters task list for the current system date)
   │   ├── 4. Update Task Status (Process: [Select Task] -> [Choose New Status: Pending/In Progress/Done])
   │   ├── 5. View All Tasks (History of all tasks)
   │   ├── 6. Delete Task (Process: [Select Task from list] -> [Confirm Removal])
   │   ├── 7. Generate Task Reports (PDF) (Generates PDF of routine)
   │   └── 0. Back to Main Menu (Return to Main Menu)
   ├── 🛒 3. Cash (Financial and expense tracking)
   │   ├── 1. Add New Buy Record (Process: [Enter Item Name] -> [Enter Price] -> [Enter Date: DD-MM-YYYY or 't'] -> [Enter Time: HH.MM + a/p or 'n'])
   │   ├── 2. Edit Buy Record (Process: [Select Record] -> [Edit Name/Price/Date/Time])
   │   ├── 3. Remove Buy Record (Process: [Select Record] -> [Confirm Deletion])
   │   ├── 4. View Pending Expense (Lists buys marked as 'Pending')
   │   ├── 5. Expense Management (Status handling)
   │   │   ├── 1. Mark Today's Expenses as Received (Automatically updates today's 'Pending' buys to 'Received')
   │   │   ├── 2. View Pending Buys (Detailed grouping of pending items by date)
   │   │   ├── 3. Mark Expense as Received (Process: [Select specific pending record] -> [Update to Received])
   │   │   ├── 4. Mark Expenses as Received by Date Range (Process: [Enter Date Range] -> [Batch updates status])
   │   │   ├── 5. View Expenses by Date (Process: [Enter Date] -> [List of Received items])
   │   │   ├── 6. View All Expenses (Full history with 'Received' total sums)
   │   │   ├── 7. Update Status to Pending (Process: [Select Received record] -> [Revert to Pending])
   │   │   ├── 8. Remove Expense (Select and Delete)
   │   │   ├── 9. Generate PDF Report (Generates Expense Statement PDF)
   │   │   └── 0. Back to Main Menu (Return to Cash Menu)
   │   ├── 6. Money Manager (Account and Fund management)
   │   │   ├── 1. Create New Fund Account (Process: [Enter Name] -> [Select Platform: BANK/BKASH/NAGAD/etc.] -> [Enter Bank Name if BANK] -> [Enter Initial Balance])
   │   │   ├── 2. Deposit Money (Add) (Process: [Select Account] -> [Enter Amount] -> [Enter Note])
   │   │   ├── 3. Withdraw/Spend Money (Delete) (Process: [Select Account] -> [Enter Amount or 'ALL'] -> [Enter Note])
   │   │   ├── 4. Transaction Statement (Reverse chronological list of transactions)
   │   │   ├── 5. Undo/Remove Last Record (Process: [Select Transaction ID] -> [Reverts account balance and removes log])
   │   │   ├── 6. Close/Remove Account (Process: [Select Account] -> [Confirm Closure])
   │   │   ├── 7. Generate PDF Report (Financial Statement PDF)
   │   │   ├── 8. Manual Balance Adjustment (Process: [Select Account] -> [Enter Correct Balance] -> [Updates without normal tx log])
   │   │   └── 0. Back to Cash Menu (Return to Cash Menu)
   │   ├── 7. View Today's Buy Records (Simplified list of today's spending)
   │   ├── 8. View All Buy Records (Full database of purchases)
   │   ├── 9. View Total Expenditure (Shows sum total of all purchases)
   │   ├── 10. Generate Daily Buy Report (PDF) (PDF export of all buys)
   │   └── 0. Back to Main Menu (Return to Main Menu)
   ├── 📚 4. View Report (Action Report Log) (Displays a chronological log of every action taken in the app)
   ├── 🔄 5. Flow (Navigation Flow History) (Visual history of menu entries and exits with depth indicators)
   ├── 📊 6. Data (data.json Content) (Recursive display of the underlying JSON database)
   ├── 📄 7. Generate Reports (PDF) (Batch report generation)
   │   ├── 1. University, Routine, Cash, Flow & Data Report (Full backup PDF)
   │   ├── 2. View Report (Action Log PDF)
   │   ├── 3. Flow (Flow History PDF)
   │   ├── 4. Data (JSON View PDF)
   │   └── 0. Back to Main Menu (Return to Main Menu)
   └── 🔚 0. Exit (Safely close the application)
